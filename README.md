# Dodge Game (Unity)

Unity로 제작한 간단한 회피 게임입니다.  
튜토리얼 수준의 시작 프로젝트를 기반으로,  
기능을 하나씩 개선하며 구조와 확장성을 고민하는 것을 목표로 개발 중입니다.

---

## 🎮 게임 설명
- 플레이어를 좌우로 이동시켜 위에서 떨어지는 장애물을 피하는 게임
- 생존 시간에 따라 점수가 증가
- 메인 씬 / 플레이 씬 분리

---

## 🛠 구현 기능
- 플레이어 이동 및 충돌 처리
- 점수 시스템
- 씬 전환 구조
- **BGM 시스템**
  - 씬별 배경음악 분리
  - 옵션 메뉴에서 BGM On/Off
  - 볼륨 슬라이더
  - BGM 토글 시 재시작되지 않고 mute/unmute 방식으로 처리

---

## 🎵 오디오 설계 포인트
- AudioManager 싱글톤 구조
- DontDestroyOnLoad를 이용한 전역 BGM 관리
- BGM 토글은 Stop이 아닌 Mute 방식으로 처리하여
  사용자가 음악을 처음부터 다시 듣지 않도록 설계
- PlayerPrefs를 이용한 설정값 저장

---

## 🧩 사용 기술
- Unity
- C#
- AudioSource / AudioMixer
- PlayerPrefs

---

## 📈 개발 의도
이 프로젝트는 완성된 게임보다는  
**기본적인 게임 구조를 직접 설계하고 개선해 나가는 과정**에 초점을 두고 있습니다.

작은 기능이라도
- 왜 이렇게 설계했는지
- 확장 시 어떤 문제가 생길 수 있는지
를 고민하며 단계적으로 발전시키고 있습니다.
